Detailed Description

template<>
struct boost::gil::image_read_info< bmp_tag >

Constant signature for bmp file format.

Read information for bmp images.

The structure is returned when using read_image_info.

Member Data Documentation

bmp_bits_per_pixel::type _bits_per_pixel

The number of bits per pixel, which is the color depth of the image. Typical values are 1, 4, 8, 16, 24 and 32.

bmp_header_size::type _header_size

The size of this header:

  • 40 bytes for Windows V3 header
  • 12 bytes for OS/2 V1 header
bmp_image_size::type _image_size

The image size. This is the size of the raw bitmap data (see below), and should not be confused with the file size.

bmp_num_important_colors::type _num_important_colors

The number of important colors used, or 0 when every color is important; generally ignored.
